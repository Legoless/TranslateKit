TranslateKit
=============

TranslateKit is a modern SDK to develop translation applications. It provides features like:

- Transifex Objective-C API wrapper based on AFNetworking 2.0 and block syntax.

List is constantly growing.

Features
=============

- Apple Strings format support
- Transifex API Wrapper
